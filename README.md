# Compiler Construction: Parsing Techniques

This project demonstrates **operator precedence parsing** and **backtracking parsing** using Python and Tkinter.

## Features
- Operator precedence table visualization
- GUI with tabs for different parsing techniques
- Step-by-step parsing actions (Stack, Input, Action)

## How to Run
1. Clone or download the repository.
2. Run `operator_precedence_parser.py` in Python 3.
3. Enter an expression (e.g., `id + id * id`) and click **Parse**.

## Requirements
- Python 3.x
- Tkinter (included with Python)
